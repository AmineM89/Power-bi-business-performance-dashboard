ğŸ§© Data Model

The data model behind this dashboard is built using a Snowflake schema with three fact tables:

  • Sales Fact Table: Contains transactional sales data, including order details, revenue, and product categories.
  • Operations Fact Table: Tracks operational metrics such as delivery times, canceled orders, and shipping costs.
  • Customer Satisfaction Fact Table: Contains customer behavior and satisfaction metrics, including customer ratings and delivery performance.

The snowflake architecture allows for efficient and scalable data processing, enabling detailed analysis across different metrics. The model uses Power Query to integrate and transform data from multiple sources, and the relationships between fact and dimension tables enable seamless filtering and slicing.
